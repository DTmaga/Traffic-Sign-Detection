# Traffic Sign Detection (OpenCV)

A computer vision project that detects traffic signs from videos or webcam streams using traditional image processing techniques without deep learning.

## Features

* Traffic sign detection from video or webcam
* HSV color segmentation
* CLAHE for image enhancement
* Morphological operations
* Canny edge detection
* Contour and shape detection
* Stable detection using temporal frame tracking

## Technologies

* Python
* OpenCV
* NumPy

## Results

The system successfully detects most common traffic signs under suitable lighting conditions while maintaining stable performance across consecutive frames.

## How to Run

```bash
pip install -r requirements.txt
python main.py
```
